# Kling UI Wiki

Welcome to the **Kling UI** documentation. Kling UI is a robust batch video generation tool designed to streamline the creation of AI videos from images using the **fal.ai Platform API**.

## Core Features

*   **Batch Processing**: Convert multiple images into high-quality 10-second AI videos simultaneously.
*   **Dual UI Modes**:
    *   **CLI Mode**: A terminal-based Rich UI menu system for efficient command-line operations.
    *   **GUI Mode**: A modern Tkinter-based drag-and-drop interface with queue management.
*   **Real-time Balance Tracking**: Optional monitoring of your fal.ai credits via background Selenium automation.
*   **Flexible Configuration**: Customizable models, prompts, output folders, and reprocessing modes.
*   **Advanced Video Effects**: Automatic ping-pong looping for seamless video displays.
